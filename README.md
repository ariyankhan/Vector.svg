# Vector.svg

A Javascript library for creating vector graphics using SVG. It uses SVG 1.1 W3C Spec 
and written in pure ES5.

## Install

    `npm install vector.svg`
    
## Build
    
    `npm run build`
     
## Test

    `npm test`
